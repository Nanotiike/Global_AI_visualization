---------------------------------------------------

This project was made as a part of Interactive Data Visualization -course by Helsinki Unversity

## Installation

1. Clone the repository

2. Create a virtual environment:

```
python -m venv venv
```

3. Activate it:

Linux/Mac: source venv/bin/activate

Windows: venv\Scripts\activate

4. Install dependencies:

```
pip install -r requirements.txt
```
5. Add path to data in .env

For all the following: PATH_TO_DATAPOINTS, PATH_TO_PARAMETERS, PATH_TO_COMPUTATION, PATH_TO_HARDWARE_AND_ENERGY, PATH_TO_PUBLICATIONS, PATH_TO_PATENTS, PATH_TO_AI_SYSTEMS, PATH_TO_FUNDING, PATH_TO_BILLS, PATH_TO_AI_INCIDENTS, PATH_TO_VIEWS

6. Start the app
```
python3 src/main.py
```
